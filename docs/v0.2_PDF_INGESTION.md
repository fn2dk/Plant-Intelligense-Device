# v0.2 PDF Ingestion

v0.2 introduces the first functional backend analysis path.

## What works

- A PDF can be uploaded to `/documents/upload`.
- The backend stores it under `uploads/{document_id}/source.pdf`.
- The analysis endpoint `/analysis/{document_id}` runs the orchestrator.
- The orchestrator extracts embedded PDF text.
- Pages are classified as P&ID, legend, component list or unknown.
- Basic engineering tags are detected from text.
- Basic legend terms are extracted from legend pages.

## What does not work yet

- No scanned drawing OCR.
- No image rendering.
- No symbol bounding boxes.
- No OpenCV line tracing.
- No real graph edges.
- No isolation calculation.

## Next sprint

v0.3 should add page rendering and coordinate-aware output.
