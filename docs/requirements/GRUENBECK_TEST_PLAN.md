# Grünbeck P&ID Test Plan

The first real test document is the uploaded Grünbeck water treatment P&ID package.

Expected document sections:

- index page
- P&ID drawing page
- component list
- legend pages

Test goals:

1. Classify legend pages.
2. Classify component list page.
3. Identify P&ID drawing page.
4. Extract visible tags such as pumps, valves, tanks, instruments and filters.
5. Detect project-specific line colour legend.
6. Prepare output for graph generation.

Success is measured by repeatable extraction, not perfect recognition in the first version.
