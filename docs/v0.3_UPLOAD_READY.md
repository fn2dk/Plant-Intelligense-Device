# v0.3 Upload Ready

This sprint produces the first local end-to-end user flow.

## User flow

1. Start backend.
2. Start frontend.
3. Open browser.
4. Upload PDF.
5. Backend stores file.
6. Backend analyzes file.
7. Frontend displays structured result.

## Current limitation

v0.3 does not yet render drawings or calculate coordinates.

## Next sprint

v0.4 will introduce PDF page rendering to PNG, page image storage and coordinate-aware outputs.
