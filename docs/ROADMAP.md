# Roadmap

## Sprint 0

Create foundation, architecture and skeleton code.

## Sprint 1

Single PDF analysis:

- Upload endpoint
- Page classification
- Basic tag extraction
- Store document metadata

## Sprint 2

Frontend viewer:

- PDF.js rendering
- Page list
- Analysis results panel
- Clickable placeholder overlays

## Sprint 3

Legend and OCR:

- Render pages as images
- PaddleOCR integration
- Vision model interface
- Legend structure extraction

## Sprint 4

Symbol and connection tracing:

- OpenCV line extraction
- Object detection abstraction
- First graph from real coordinates

## Sprint 5

Component register:

- Excel upload
- Tag matching
- Missing/duplicate/conflict reports

## Sprint 6

Isolation engine:

- Mechanical isolation proposal
- Boundary highlighting
- Human verification workflow

## Sprint 7

Electrical profile:

- SLD object types
- Feeder tracing
- Breaker/cable/motor graph
