## Test Cases

### TC-001: Sensor Data Collection
**Objective:** Verify that sensor data is correctly collected and stored.
- **Steps:**
  1. Connect sensor to plant
  2. Record environmental readings
  3. Verify data in database
- **Expected Result:** All sensor readings stored accurately

### TC-002: AI Health Assessment
**Objective:** Verify AI engine correctly assesses plant health.
- **Steps:**
  1. Load known plant conditions
  2. Run AI analysis
  3. Compare recommendations with expected output
- **Expected Result:** AI provides accurate health status

### TC-003: Frontend Display
**Objective:** Verify frontend correctly displays plant data.
- **Steps:**
  1. Navigate to plant dashboard
  2. Verify data matches database
  3. Check visualization accuracy
- **Expected Result:** Frontend displays current and historical data correctly

### TC-004: Alert System
**Objective:** Verify system alerts user to critical conditions.
- **Steps:**
  1. Set plant to critical condition
  2. Verify alert is generated
  3. Verify user notification sent
- **Expected Result:** User receives timely alerts
